Gmail lié à un compte Microsoft — procédure

C'est la situation la plus délicate. Suis ces étapes dans l'ordre exact.

  1. Ouvre une fenêtre de navigation privée (Ctrl+Maj+N)
  2. Va sur login.microsoftonline.com
  3. Clique « Se déconnecter de tous les comptes »
  4. Entre ton adresse Gmail
  5. Choisis « Envoyer un code de vérification » — ne choisis PAS ton compte Microsoft personnel
  6. Entre le code reçu dans ta boîte Gmail
  7. Utilise ensuite le lien direct vers MMM+ Agora

Vider le cache du navigateur

Le cache peut conserver de vieilles sessions et causer des problèmes de connexion.

  1. Chrome / Edge : Ctrl+Maj+Suppr → Depuis toujours → Cookies + Images en cache → Effacer
  2. Firefox : Ctrl+Maj+Suppr → Tout → Cookies + Cache → Effacer
  3. Safari Mac : Menu Safari → Préférences → Confidentialité → Gérer les données → Tout supprimer
  4. Safari iPhone/iPad : Réglages → Safari → Effacer historique et données
  5. Après avoir vidé le cache, reconnecte-toi via le lien direct MMM+ Agora
Contacter le support →
