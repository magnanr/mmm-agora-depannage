Normal pour la première fois — ne se répétera pas

Lors de ta première visite depuis ce navigateur, Microsoft doit valider ta session sur Viva Engage séparément de SharePoint. C'est un comportement B2B standard.

  1. Entre ton code OTP ou approuve la notification Microsoft Authenticator
  2. Coche « Ne plus me demander pendant 30 jours » si l'option est disponible
  3. Lors de ta prochaine visite sur le même appareil, tu n'auras plus cette étape
Boucle de session — conflit de comptes

Tu es probablement connecté simultanément à plusieurs comptes Microsoft (MMM+ + employeur + personnel). Le navigateur ne sait plus lequel choisir.

  1. Ouvre une fenêtre de navigation privée (Ctrl+Maj+N ou Cmd+Maj+N)
  2. Accède à MMM+ Agora via le lien de ton courriel d'invitation
  3. Connecte-toi uniquement avec le courriel de ton compte MMM+
  4. Si ça fonctionne, bookmarke cette URL et utilise toujours la fenêtre privée pour MMM+ Agora
  5. Si ça boucle encore, contacte le support

Mauvais espace — utilise le bon compte

Microsoft t'a connecté avec ton compte d'employeur au lieu de ton compte MMM+. C'est fréquent chez les membres avec un compte professionnel actif (CUSM, enseignants, etc.).

  1. Déconnecte-toi de ton compte d'employeur dans le navigateur
  2. Ouvre une fenêtre de navigation privée
  3. Connecte-toi avec le courriel de ton compte MMM+
  4. Solution permanente : crée un profil de navigateur dédié pour MMM+ Agora (Chrome et Edge le permettent)
Contacter le support →
Femmes de rivière — accès sur demande

Cette communauté est réservée aux femmes membres de MMM+. L'accès ne se fait pas automatiquement à l'inscription.

  1. Contacte le support en précisant que tu souhaites rejoindre Femmes de rivière
  2. La gestionnaire de communauté validera ta demande
  3. Tu recevras un avis une fois l'accès accordé
Contacter le support →
Aucune communauté visible — compte non provisionné

Ton compte a été créé mais n'a pas été ajouté aux groupes des communautés Viva Engage. C'est une étape distincte de l'invitation initiale.

  1. Contacte le support en indiquant ton adresse courriel MMM+
  2. L'administrateur corrigera l'appartenance de ton compte aux 8 communautés
  3. Une fois corrigé, actualise la page Agora (F5) — les communautés apparaîtront
Contacter le support →
Communautés partielles — rejoindre manuellement

Certaines communautés s'affichent seulement après les avoir rejointes manuellement.

  1. Dans Agora, clique sur Découvrir des communautés (icône grille en haut)
  2. Recherche les communautés manquantes : Rivières & Sorties, Saumon & Grandes Rivières, Artisans de la rivière, Conditions & Territoires, Carnet de pêche
  3. Clique Rejoindre pour chacune
  4. Si une communauté n'apparaît pas dans la recherche, contacte le support

Conflit avec compte d'employeur

Microsoft t'authentifie automatiquement avec ton compte professionnel dominant (CUSM, école, etc.) au lieu de ton compte MMM+.

  1. À la page de connexion, clique sur « Utiliser un autre compte »
  2. Entre le courriel de ton compte MMM+ (celui utilisé dans Yapla)
  3. Solution durable : crée un profil de navigateur dédié pour MMM+ Agora — Chrome et Edge permettent des profils séparés qui mémorisent le bon compte
  4. Si ton employeur bloque les connexions B2B externe, utilise ton courriel personnel pour ton compte MMM+
Contacter le support →
Double compte MMM+ — contact admin requis

Si tu as un compte instructeur interne (@moucheursmontrealmetro.ca) et un compte membre invité (Gmail, etc.), les deux coexistent et créent des conflits d'accès.

  1. Contacte le support en précisant tes deux adresses courriel
  2. L'administrateur identifiera le compte principal et désactivera le conflit
  3. En attendant, utilise toujours l'adresse @moucheursmontrealmetro.ca si tu es instructeur

Invitation introuvable — relance nécessaire

L'invitation (expéditeur : invites@microsoft.com) est peut-être expirée ou bloquée. Les invitations expirent après 30 jours.

  1. Contacte le support en indiquant ton adresse courriel exacte
  2. L'administrateur renverra une nouvelle invitation depuis Entra ID
  3. Accepte-la dans les 48 h
Contacter le support →
Fournisseur qui bloque les courriels Microsoft

Certains fournisseurs québécois (Videotron notamment) bloquent ou filtrent les courriels d'invitation Microsoft. Le message ne te parviendra jamais via cette adresse.

  1. Contacte le support en indiquant une adresse courriel alternative (Gmail, Outlook, etc.)
  2. L'administrateur créera ton compte avec cette nouvelle adresse
  3. Mets aussi à jour ton courriel dans ton profil Yapla pour assurer la cohérence
Contacter le support →
Vérification du courriel dans Yapla

L'invitation est envoyée à l'adresse enregistrée dans Yapla. Si elle est incorrecte, tu ne peux pas la recevoir.

  1. Connecte-toi à yapla.com avec ton adresse habituelle
  2. Vérifie le courriel dans ton profil membre
  3. Si incorrect, mets-le à jour puis contacte le support pour une nouvelle invitation

  1. Contacte le support avec ton adresse courriel
  2. Précise que ton lien est expiré

Page de consentement — c'est normal

Microsoft demande ton accord pour que MMM+ accède à ton profil de base. Étape obligatoire, une seule fois.

  1. Lis les permissions (lecture de profil et appartenance aux groupes)
  2. Clique Accepter
  3. Tu es redirigé vers MMM+ Agora automatiquement
  4. Si tu arrives sur une page blanche, actualise (F5)
Contacter le support →
URL dans le mauvais champ

Le champ demande ton adresse courriel, pas le lien du site Agora. L'app a mis en cache cette erreur — une simple correction ne suffit pas.

  1. Désinstalle complètement l'app Viva Engage
  2. Redémarre ton téléphone (vide le cache)
  3. Réinstalle depuis l'App Store ou Google Play
  4. À la première ouverture, entre uniquement ton adresse courriel MMM+

  2. Appuie sur Next
  3. Entre ensuite ton adresse courriel MMM+
  4. Connecte-toi et approuve le MFA si demandé

App bloquée — réinstallation complète

Un bug de cache empêche l'app de charger l'écran de connexion correctement.

  1. Désinstalle l'app Viva Engage
  2. Redémarre ton téléphone
  3. Réinstalle depuis l'App Store ou Google Play
  4. Entre uniquement ton adresse courriel à la première ouverture
  5. Alternative : utilise Safari ou Chrome mobile avec le lien de ton invitation
